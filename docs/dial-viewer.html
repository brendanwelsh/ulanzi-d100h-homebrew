Connect dial (WebHID) reads the real D100H — turn the physical knob and this one spins. Chrome/Edge only; pick Dial_Lite when prompted. Calibrate walks each control so the page learns your unit's codes (saved in your browser). Caveats (the device's):

  • The dial's rotate is a real volume key, so connecting will also move your system volume.
  • Over WebHID only the dial + 3 media keys are readable (Windows hides Ctrl-C/V/Z/Y). For all 7, forward events from a UlanziDeck plugin to the WebSocket box (see plugin-sdk.md).
