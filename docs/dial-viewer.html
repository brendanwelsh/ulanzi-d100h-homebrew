Connect dial (WebHID) reads the real D100H, so turning the physical knob spins this one. Chrome or Edge only; pick Dial_Lite when prompted. Calibrate walks each control so the page learns your unit's codes (saved in your browser). Caveats (the device's):

  • Close Ulanzi Studio first. While it's running the dial is in online mode (Studio handles the input), so the page won't see the default volume/media codes it listens for.
  • The dial's rotate is a real volume key, so connecting will also move your system volume.
  • Over WebHID only the dial and 3 media keys are readable (Windows hides Ctrl-C/V/Z/Y). For all 7, forward events from a UlanziDeck plugin to the WebSocket box (see plugin-sdk.md).
  • VID 0xfff1 / PID 0x0082 (see hardware.md). Assets and method: dial-skin.md.
