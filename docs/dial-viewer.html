Connect dial (WebHID) reads the real D100H directly — turn the physical knob and this one spins. Chrome/Edge only; pick Dial_Lite when prompted. Caveats (the device's, not ours):

  • The dial's rotate is a real volume key, so connecting will also move your system volume.
  • Only the dial + 3 media keys are readable this way (Windows hides the Ctrl-C/V/Z/Y keys). For all 7 keys cleanly, use a UlanziDeck plugin that forwards events to the WebSocket box above (see plugin-sdk.md).
  • VID 0xfff1 / PID 0x0082 — see hardware.md. Assets + method: dial-skin.md.
